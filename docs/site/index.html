Au Jour Le Jour

Stay on the essentials. Take it one day at a time.

A local-first essentials ledger that keeps you focused on what must be paid, what’s already handled, and what you need to earn per day to stay stable.

Open the App Install locally

Local-first by design

Everything stays on your device. No cloud, no accounts, no tracking.

Essentials only

Track the bare minimum to avoid lifestyle creep and stay disciplined.

Agent-ready

Built as an Essentials Ledger service to integrate with MyCasaPro later.

Install & Run Locally

  1. Clone the repo
  2. Run npm install
  3. Start with npm run start

Local data stays on your machine. Use Export/Import to move data.

What it does

  • Generates monthly bill instances from your templates
  • Tracks paid, partial, pending, and skipped items
  • Calculates required, paid, remaining, and daily/weekly need
  • Shows overdue and due-soon lists automatically
  • Supports sinking funds for non-monthly expenses
